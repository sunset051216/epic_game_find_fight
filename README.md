# 潮蚀荒原：第三人称搜打撤游戏

《潮蚀荒原》是一款基于 Unreal Engine 5.7 制作的 3D 第三人称单机搜打撤课程项目。游戏以“荒漠时代”和“海平面上涨后的废墟探索”为背景，玩家需要在限定时间内进入危险区域，探索宝箱、击杀怪物、收集金币，并在倒计时结束前成功撤离。

## 一、项目基本信息

- 游戏类型：3D 第三人称搜打撤
- 游戏平台：PC
- 网络模式：单机
- 玩家人数：1 人
- 单局时长：5 分钟
- 胜利条件：收集金币并通过撤离点离开
- 开发引擎：Unreal Engine 5.7
- 项目名称：ExtractionGame

## 二、世界观背景

游戏发生在名为“潮蚀荒原”的末世时代。冰山融化、海平面上涨，大量旧城市与港口废墟被海水吞没。金币成为荒原社会中最重要的交易标识，探索者们乘船前往被淹没的城市废墟寻找金币。

但金币聚集之处往往伴随着危险。荒漠时代导致生物发生异变，怪物盘踞在宝箱、金库和撤离路线附近。玩家需要在有限时间内完成搜索、战斗与撤离。

## 三、游戏流程

完整游戏流程如下：

```text
主界面
→ 准备探索界面
→ 地图选择 / 角色选择
→ 加载界面
→ 进入游戏地图
→ 探索场景
→ 打开宝箱获得金币
→ 与怪物战斗
→ 倒计时后撤离点出现
→ 成功撤离 / 死亡 / 超时
→ 结算界面
→ 返回主界面
```

## 五、角色系统

游戏中加入了角色选择系统。不同角色具有不同属性，会影响生命值、攻击力、移动速度、开箱速度和金币收益。

| 角色 | 定位 | 生命 | 攻击 | 移动速度 | 开箱耗时 | 金币加成 |
| --- | --- | ---: | ---: | ---: | ---: | ---: |
| 废港护卫 | 高生命、高攻击，适合正面战斗 | 130 | 25 | 600 | 2.0 秒 | 0% |
| 潮金拾荒者 | 移动速度快，适合快速搜刮和撤离 | 90 | 15 | 750 | 1.5 秒 | 0% |
| 遗迹工程师 | 开箱最快，并拥有金币收益加成 | 100 | 12 | 600 | 1.2 秒 | 10% |

角色选择结果会通过 `BP_GameInstance` 保存，并在进入地图后应用到玩家角色身上。

## 六、地图系统

项目中制作了地图选择流程，并设计了三张地图：

### 1. 沉没旧港遗址

主地图。旧港废墟被海水侵蚀，场景中包含地面区域、上层结构、宝箱、怪物和撤离点。该地图用于展示课程要求中的核心玩法。

### 2. 地铁潮蚀金库

扩展地图。地铁通道因地下水冲刷出现裂缝，通往隐藏金库区域。设计方向包括地下多层结构、高价值宝箱和强力怪物镇守。

### 3. 荒漠信号废城

扩展地图。玩家出生在屋顶区域，只能通过建筑之间的桥梁和通道移动。设计方向包括屋顶探索、机关按钮和信号塔撤离。

## 七、核心玩法系统

### 宝箱与金币

- 玩家靠近宝箱后开始打开流程。
- 宝箱打开需要一定时间。
- 打开成功后获得随机金币。
- 不同角色会影响开箱耗时。
- 遗迹工程师拥有额外金币收益加成。
- 已开启宝箱会计入结算统计。

### 怪物 AI

怪物使用行为树和黑板实现基础 AI 行为，包括：

- 巡逻
- 发现玩家
- 追击玩家
- 攻击玩家

怪物攻击会扣除玩家生命值。玩家击杀怪物后会获得金币奖励，并提升战斗力。

### 战斗力成长

玩家每击杀一个怪物，战斗力增加。实际伤害会随战斗力提升：

```text
实际伤害 = 基础攻击 + 战斗力 * 5
```

### 撤离与失败

- 游戏倒计时为 5 分钟。
- 撤离点会在后期出现。
- 玩家触碰撤离点后成功逃生。
- 玩家生命值归 0 时死亡。
- 倒计时结束仍未撤离则超时失败。
- 死亡和超时会损失部分金币。

## 十一、运行方式

1. 安装 Unreal Engine 5.7 或兼容版本。
2. 打开 `ExtractionGame.uproject`。
3. 进入项目后运行游戏。
4. 从主界面点击“开始游戏”，进入准备探索流程。
5. 选择地图与角色后点击“开始探索”。
