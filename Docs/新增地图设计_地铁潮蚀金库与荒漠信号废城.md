# 新增地图设计：地铁潮蚀金库与荒漠信号废城

## 地图二：地铁潮蚀金库

地图文件：

```text
LV_TideVault
```

定位：

```text
地下探索 / 高风险高收益 / Boss 金库
```

### 背景设定

荒漠时代后，废弃地铁站的地下结构被混乱地下水冲毁。水流改变了原本的轨道线路，并冲开了一条通往旧世界隐藏金库的裂缝。

玩家从地表地铁站进入，穿过站台、隧道和地下水裂缝，最终抵达被 Boss 镇守的深层金库。

### 地图结构

```text
地表层：废弃地铁站入口
B1：地铁大厅 / 站台 / 半淹没轨道 / 列车残骸
B2：坍塌地铁隧道 / 地下水裂缝 / 水泵房 / 断裂轨道桥
B3：隐藏运金金库 / 保险柜走廊 / Boss 大金库
```

### 关键设计

```text
地表有自然光和暖色光源
地表入口是单体建筑，不通往二层，只能向下
越往地下越暗
B1 有残留冷光
B2 有蓝色水光
B3 有金色和紫色 Boss 光
Boss 大金库内有高价值宝箱
```

### 撤离点设计

地铁返回撤离点：

```text
位置：B1 附近
特点：较安全，收益较低
```

深层维修升降梯：

```text
位置：B3 Boss 金库附近
特点：危险，但靠近高价值宝箱
```

### 玩法节奏

```text
出生在地表地铁站附近
进入地铁大厅
搜索站台和列车残骸
穿过坍塌隧道和地下水裂缝
进入隐藏金库
挑战 Boss 区域
选择安全撤离或深层撤离
```

### 地图选择介绍

```text
地铁潮蚀金库

难度：困难
类型：地下探索 / Boss 金库

废弃地铁站的地下水冲开了通往旧世界金库的裂缝。
地表区域光线充足，越深入地下越黑暗，收益也越高。
深层金库中存在 Boss 和高价值宝箱。
```

## 地图三：荒漠信号废城

地图文件：

```text
LV_SignalRuins
```

定位：

```text
屋顶连桥 / 高空路线规划 / 机关解锁 / 摔落死亡
```

### 背景设定

旧城市被海水和风沙侵蚀后，地面区域被盐雾和怪物占据。幸存者只能通过废楼屋顶、内部楼梯和楼间连桥移动。

城市最高处的信号塔仍保留撤离信标，但前往信号塔的最终路线被锁定，玩家必须先找到控制按钮。

### 地图结构

```text
500m×500m 屋顶城市
25 栋建筑
每栋建筑至少 3 层
部分关键建筑 4-6 层
玩家出生在屋顶
地面为死亡区 / 盐雾区
```

### 路线规则

```text
玩家不能走地面
摔落到地面会死亡
只能通过屋顶、二层/上层内部、楼间桥梁移动
每栋建筑至少有 1 条通往其他建筑的路线
约 30% 建筑拥有额外第二条连接路线
每栋建筑有内部楼梯，但不通往一层地面
```

### 关键区域

SpawnHouse：

```text
玩家出生屋顶
```

ButtonBuilding：

```text
机关按钮建筑
玩家需要找到按钮，解锁通往最终信号塔的路线
```

TowerGate：

```text
最终路线前的过渡建筑
```

SignalTower：

```text
最高建筑
唯一撤离点位于顶层
```

FinalGuardRoof：

```text
最终守卫区域
靠近信号塔撤离点
```

### 撤离点设计

```text
唯一撤离点：最高信号塔屋顶
```

### 机关设计

按钮未激活：

```text
最终桥梁 / 通路被阻挡
```

按钮激活后：

```text
通往信号塔的最终路线开启
玩家可以前往最高塔顶撤离
```

### 玩法节奏

```text
出生在屋顶
沿楼间桥梁移动
进入不同建筑的上层内部
搜集金币和宝箱
寻找机关按钮建筑
按下按钮
解锁最终信号塔通路
通过守卫区域
到达最高塔顶撤离
```

### 地图选择介绍

```text
荒漠信号废城

难度：困难
类型：高空机关地图

玩家出生在废城屋顶，地面被盐雾和怪物占据。
若坠落至地面将直接死亡。
玩家必须通过楼顶桥梁和建筑内部路线寻找控制按钮，
解锁通往最高信号塔的道路，并从塔顶撤离。
```
