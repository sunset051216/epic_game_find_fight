# 《潮蚀荒原》地图设计：沉没旧港遗址

## 地图定位

这是一个 100m x 100m 的第三人称搜打撤灰盒地图。主题是“荒漠时代中被海平面吞没过的旧港城市遗址”：沙化码头、浅水废墟、集装箱、断桥、旧银行/金库入口、高处信号塔共同组成探索区域。

地图目标是支撑 5 分钟单局流程：

进入旧港 -> 搜索宝箱 -> 遭遇怪物 -> 继续深入或撤退 -> 第 4 分钟撤离点出现 -> 触碰撤离点结算。

## 尺寸

- 地图总尺寸：10000 x 10000 Unreal Units，也就是 100m x 100m。
- 地面中心：世界原点附近。
- 可探索范围：X/Y 约 -5000 到 5000。
- 至少一层地面 + 一层上层结构，满足作业要求。

## 区域划分

1. **南侧临时登陆点**
   - 玩家出生区域。
   - 有破旧码头和船只停靠暗示。
   - 难度低，放 1-2 个宝箱。

2. **中部沙化广场**
   - 主探索区。
   - 有集装箱、废墙、浅水沟。
   - 放置多条路线，避免直线通关。

3. **东侧沉没街区**
   - 有浅水区域和半淹没建筑。
   - 宝箱较多，怪物点较密。

4. **西侧废弃金库入口**
   - 主题资源点。
   - 放置精英怪/高价值宝箱的预留位置。

5. **北侧二层平台与信号塔**
   - 上层结构区域。
   - 通过楼梯到达。
   - 可以作为第二逃生点。

## 点位数量

### 宝箱点，至少 10 个

建议点位：

1. 登陆点左侧木箱区
2. 登陆点右侧废船旁
3. 中央集装箱后
4. 中央浅水沟旁
5. 西侧金库入口
6. 西侧断墙后
7. 东侧沉没街道
8. 东侧高台下
9. 北侧楼梯平台
10. 北侧信号塔底部

### 怪物刷新点，至少 5 个

建议点位：

1. 中央广场巡逻点
2. 西侧金库守卫点
3. 东侧浅水废墟点
4. 北侧平台下方
5. 北侧信号塔区域

### 逃生点，至少 2 个

1. **码头撤离点**
   - 南侧/东南侧近水区域。
   - 表现为撤离船或蓝色光柱。

2. **信号塔撤离点**
   - 北侧上层平台。
   - 表现为高处信标或传送门。

## 游戏性路线

玩家出生在南侧，初始安全。越往北/东西两侧探索，宝箱价值感越高，怪物压力越强。逃生点在第 4 分钟出现后，一个在低处码头，一个在高处信号塔，玩家需要根据当前位置选择最近路线。

## 灰盒视觉风格

- 沙化地面：大面积浅黄色平面。
- 浅水区域：低矮蓝色透明/蓝色平面。
- 集装箱：长方体障碍。
- 废墙：不同高度的窄长方体。
- 上层平台：北侧二层平台 + 楼梯。
- 信号塔：细长竖向结构，后续可加发光材质或 Niagara 特效。
